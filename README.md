# Weapon Balls

2D top-down игра на Unity. Проект для НИС «Разработка игр на Unity», ФКН ВШЭ.

## Описание

Weapon Balls — аркадная 2D игра в средневековом сеттинге. Игрок управляет рыцарем (шар с вращающимся мечом), сражается с врагами на арене и продвигается к тронному залу, чтобы свергнуть кровавого короля.

Игра состоит из 3 уровней с прокачкой между ними. Враги — шары разных типов, которые отскакивают от стен и наносят урон при контакте или оружием.

## Управление

- **WASD** — движение
- **ESC** — пауза
- Оружие вращается автоматически вокруг персонажа, наносит урон при касании врагов

## Уровни

1. **Улица** — 5 обычных рыцарей на булыжной мостовой
2. **Замок** — 3 обычных рыцаря + 1 тяжёлый с мечом
3. **Тронный зал** — Король (стреляет фаерболами) + 2 тяжёлых рыцаря

Между уровнями — экран прокачки, можно улучшить урон или здоровье за золото (выпадает из врагов).

## Как запустить

1. Открыть проект в Unity (2022.3+)
2. Убедиться что в `Assets/` лежат папки `Scripts/`, `Editor/`, `Textures/`, `Audio/`, `Fonts/`
3. В меню Unity: **WeaponBalls → Setup Project**
4. Дождаться сборки (создаст сцены, префабы, материалы, SO)
5. Нажать **Play**

При повторной сборке — удалить `Assets/Prefabs/`, `Assets/Scenes/`, `Assets/ScriptableObjects/`, `Assets/Resources/` перед Setup Project.

## Сцены

- **MainMenu** — замок на весь экран, кнопки Играть/Выход, покачивание
- **Story** — вступительный текст на фоне горящей деревни
- **Game** — основной геймплей, 3 уровня подряд
- **Epilogue** — финальный текст на фоне тронного зала

## Типы врагов

| Тип | HP | Урон | Скорость | Особенности |
|---|---|---|---|---|
| Рыцарь | 6 | 1 | 7.5 | Урон при столкновении |
| Тяжёлый рыцарь | 20 | 2 | 3.5 | Вращающийся меч |
| Король | 12 | 3 | 2.5 | Стреляет 8 фаерболами каждые 3 сек |

## Прокачка

- Урон +1 за 2 золотые монеты
- Здоровье +3 за 2 золотые монеты
- Лечилка восстанавливает 60% от максимального HP

## Баланс игрока

- Начальное HP: 10
- Начальный урон: 2
- Скорость: 8

## Звуки

Музыка зациклена на каждой сцене, при прокачке приглушается. Звуковые эффекты — удары по разным типам врагов, подбор монет/лечилок, фаерболы, нажатие кнопок.

## Особенности реализации

- Весь проект собирается одной кнопкой через `ProjectAutoSetup.cs`
- Физика: top-down (гравитация = 0), враги отскакивают от стен (bounciness = 1)
- Арена 28x28, стены со static Rigidbody2D
- Данные врагов создаются в рантайме (WaveManager.EnsureEnemyData) для надёжности
- Аудио загружается из Resources через AudioManager (DontDestroyOnLoad)
- Фон за ареной меняет цвет в зависимости от уровня

## Требования

- Unity 2022.3 или новее
- Платформа: Windows / Mac / Linux
